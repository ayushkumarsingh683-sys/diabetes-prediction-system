# diabetes-prediction-system
# 🩺 AI-Driven Clinical Disease Prediction System

An end-to-end Machine Learning web application that predicts diabetes risk based on patient clinical metrics using an ensemble of classification models.

## 📊 Model Performance Leaderboard
* **Random Forest Classifier: 95.00% Accuracy (Winner)🏆**
* Logistic Regression: 77.92% Accuracy
* Support Vector Machine (SVM): 75.32% Accuracy
* XGBoost Classifier: 74.03% Accuracy

## 🛠️ Tech Stack Used
* **Language:** Python
* **Machine Learning:** Scikit-Learn, XGBoost
* **Web Interface:** Streamlit
* **Data Handling:** Pandas, NumPy
